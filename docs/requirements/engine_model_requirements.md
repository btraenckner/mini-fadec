EMR-001:
The engine model shall represent the rotor speed between
39,000 rpm and 128,000 rpm.

EMR-002:
The engine model shall represent exhaust gas temperature.

EMR-003:
The engine model shall use a normalized fuel command as input.

EMR-004:
The engine model shall estimate thrust from rotor speed.

EMR-005:
The initial model shall require only rotor-speed and exhaust-
temperature feedback for closed-loop control.

EMR-006:
The initial model shall represent running operation only;
start and shutdown sequences are excluded.
