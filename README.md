# Mini-FADEC

Mini-FADEC is an educational embedded control platform for learning modern software engineering, simulation, embedded systems and gas turbine control.

## Objectives

- Learn Git and professional software development
- Build dynamic system simulations
- Develop reusable control algorithms
- Implement embedded software on STM32
- Build a modular FADEC-like architecture
- Develop a hardware-in-the-loop environment
- Interface with a stationary small gas turbine
